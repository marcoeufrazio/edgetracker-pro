# EdgeTracker-Pro agent instructions

## Objetivo
Construir o módulo 1 do EdgeTracker-Pro em Python.

## Âmbito do módulo 1
Implementar apenas:
- equity curve
- drawdown absoluto
- drawdown percentual
- max drawdown
- ulcer index
- MAR ratio
- capital targets
- semáforo red/yellow/green
- resumo de saúde da conta

## Regras
- Usar Python 3.11+
- Código simples e modular
- Funções pequenas e testáveis
- Criar testes unitários
- Não criar frontend
- Não criar base de dados
- Não criar API neste módulo
- Usar os ficheiros em docs/ como fonte principal da lógica

## Done when
- Os ficheiros analytics/*.py existirem
- Os testes em tests/ passarem
- Houver um script em scripts/ para testar um exemplo
