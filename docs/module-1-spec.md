# Module 1 Spec

## Objetivo
Transformar as fórmulas do Excel em funções Python.

## Inputs
O módulo deve aceitar:
- initial_balance
- pnl_values
- cycle_target

## Outputs
O módulo deve devolver:
- equity
- drawdown_abs_series
- drawdown_pct_series
- max_drawdown_abs
- max_drawdown_pct
- ulcer_index
- mar_ratio
- green_target
- traffic_light
- health_summary

## Regras
- O módulo não pode depender de Excel
- O módulo não pode depender de base de dados
- O módulo deve ser testável
- As funções devem ser simples
