# Module 11 Spec

## Objetivo
Melhorar o visual e UX do dashboard EdgeTracker-Pro.

## Melhorias

### Cards métricos
Transformar métricas principais em cards visuais consistentes.

### Layout
Organizar o dashboard em secções claras:

- Trade Intelligence
- Strategy Analyzer
- Risk Metrics
- Health Score
- Trade Explorer

### Tipografia
Melhorar títulos e subtítulos.

### Cores
Usar cores semânticas:

- verde → valores positivos
- vermelho → valores negativos
- cinza → neutro

### Tabela
Melhorar visual da trade table:
- zebra rows
- destaque para wins/losses

### Regras
- não alterar cálculos
- apenas melhorar layout
- manter compatibilidade com Streamlit
