# Excel Mapping

## Elementos do Excel a replicar

### 1. Equity curve
A equity resulta do saldo inicial mais a soma acumulada dos resultados.

### 2. Drawdown
O drawdown é a diferença entre o pico máximo atingido e a equity atual.

### 3. Drawdown percentual
O drawdown percentual é:
(peak - equity) / peak * 100

### 4. Meta zona verde
A meta da zona verde é:
cycle_target * 1.5

### 5. Semáforo
- Se lucro atual < 60% da meta verde = red
- Se lucro atual < 100% da meta verde = yellow
- Se lucro atual >= 100% da meta verde = green
